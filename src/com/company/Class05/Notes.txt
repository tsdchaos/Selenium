------------------Radio Buttons-----------------
We can select one radio button at a time

isDisplayed--> a boolean method that tells us if a particular Web Element is on the page or not

isEnabled--> a boolean method
    - Checks if you can select that radio button or not

isSelected--> a boolean method
    - Checks if a particular button is selected or not

1. SIngle Select

    Method 2:
